# 간단한 Spring 게시판 프로젝트

# 소개 
- 이 프로젝트는 Spring 프레임워크와 Thymeleaf 템플릿 엔진,
MyBatis를 사용하여 구현된 간단한 게시판입니다.
사용자는 게시물을 조회, 작성, 수정할 수 있습니다.
또한 간단하게 세션으로 로그인 기능을 구현해보았으며,
검증 로직과 입력값 유지 기능도 구현해보았습니다.
